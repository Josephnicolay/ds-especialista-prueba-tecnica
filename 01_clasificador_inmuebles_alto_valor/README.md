# Clasificador de Inmuebles de Alto Valor

## Descripción
Modelo clasificador para identificar grupos de inmuebles de alto valor.

## Estructura del Proyecto

```
01_clasificador_inmuebles_alto_valor/
├── data/
│   ├── raw/          # Datos originales sin procesar
│   ├── processed/    # Datos procesados y limpios
│   └── external/     # Datos de fuentes externas
├── notebooks/        # Jupyter notebooks para análisis exploratorio
├── src/              # Código fuente del proyecto
├── models/           # Modelos entrenados y serializados
└── docs/             # Documentación adicional del proyecto
```

## Objetivo
Desarrollar un modelo de clasificación que permita identificar y categorizar inmuebles de alto valor basándose en características relevantes del mercado inmobiliario.

## Tareas Principales
1. Análisis exploratorio de datos (EDA)
2. Preprocesamiento y limpieza de datos
3. Ingeniería de características
4. Entrenamiento de modelos de clasificación
5. Evaluación y selección del mejor modelo
6. Documentación de resultados
