# Clasificador de Sentimientos - Texto

## Descripción
Diseño de un clasificador de texto para predecir sentimientos utilizando técnicas tradicionales de NLP y Machine Learning.

## Estructura del Proyecto

```
02_clasificador_sentimientos_texto/
├── data/
│   ├── raw/          # Datos de texto originales
│   ├── processed/    # Datos procesados y vectorizados
│   └── external/     # Datasets adicionales o de referencia
├── notebooks/        # Jupyter notebooks para análisis y experimentación
├── src/              # Código fuente del proyecto
├── models/           # Modelos entrenados y vectorizadores
└── docs/             # Documentación del proyecto
```

## Objetivo
Desarrollar un clasificador de sentimientos usando técnicas tradicionales de procesamiento de lenguaje natural (TF-IDF, Bag of Words, etc.) y algoritmos de machine learning clásicos.

## Tareas Principales
1. Exploración y análisis del corpus de texto
2. Preprocesamiento de texto (limpieza, tokenización, etc.)
3. Vectorización de texto (TF-IDF, Count Vectorizer, etc.)
4. Entrenamiento de modelos de clasificación
5. Evaluación y comparación de modelos
6. Análisis de resultados y métricas de rendimiento
