# Recursos Compartidos

## Descripción
Carpeta para código y configuraciones compartidas entre los diferentes proyectos de la prueba técnica.

## Estructura

```
shared/
├── utils/     # Funciones y utilidades comunes
└── config/    # Archivos de configuración compartidos
```

## Contenido

### utils/
Contiene funciones y utilidades reutilizables en los diferentes proyectos:
- Funciones de visualización
- Métricas de evaluación personalizadas
- Utilidades de preprocesamiento
- Funciones helper para manejo de datos

### config/
Archivos de configuración compartidos:
- Configuraciones de entorno
- Parámetros comunes
- Rutas y constantes del proyecto
